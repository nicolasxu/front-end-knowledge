boolean attributes: disabled, required, selected, checked, readOnly , and open
note 1: Their presence as attribute in HTML tag means true, their absence means false.
note 1: Their value is ignored by browser, e.g. The value of disabled attribute is ignored by browser according W3C standard.
For all the boolean attribute, there is ng- directive for them. Intepolation won't work, because if attribute is present, the value is always true.
