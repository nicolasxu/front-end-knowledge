the difference between .filter() and .find() .filter() - filter currently selected set detail: Reduce the set of matched elements to those that match the selector or pass the function's test. .find() - match the descendants of selected set Get the descendants( only descendants, not itself) of each element in the current set of matched elements, filtered by a selector, jQuery object, or element. Notes: .find() requires to pass in an expression as selector pass in "*" to select all.
