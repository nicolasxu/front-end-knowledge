Description: Selects all button elements and elements of type button. An equivalent selector to $( ":button" ) using valid CSS is $( "button, input[type='button']" ). Because :button is a jQuery extension and not part of the CSS specification, queries using :button cannot take advantage of the performance boost provided by the native DOM querySelectorAll() method. To achieve the best performance when using :button to select elements, first select the elements using a pure CSS selector, then use .filter(":button"). button demo
